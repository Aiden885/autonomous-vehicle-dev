◆ pcdAffinityPropagationCluster()

void lidarTools::pcdAffinityPropagationCluster ( pcl::PointCloud< pcl::PointXYZI >::Ptr  input,
std::vector< pcl::PointIndices > &  cluster_indices,
double  damping,
int  max_iter 
)

Performs Affinity Propagation clustering on a point cloud.

Implements the Affinity Propagation algorithm which:

  1. Computes pairwise similarities between points
  2. Iteratively updates responsibility and availability matrices
  3. Identifies exemplars (cluster centers) through message passing
  4. Assigns points to their nearest exemplar to form clusters
参数
[in]inputInput point cloud to be clustered
[out]cluster_indicesVector containing point indices for each cluster
[in]dampingDamping factor (0-1) to avoid numerical oscillations
[in]max_iterMaximum number of iterations for convergence
注解
The algorithm works by:
  1. Building similarity matrix based on negative squared distances
  2. Iteratively updating responsibility and availability matrices
  3. Applying damping to avoid oscillations
  4. Identifying exemplars where responsibility + availability > 0
  5. Assigning points to nearest exemplar to form final clusters

注解
The algorithm works by:
  1. For each point, calculating weighted mean of nearby points within bandwidth
  2. Shifting the point towards the weighted mean
  3. Repeating until convergence or max iterations reached
  4. Points that converge to the same mode are grouped into clusters
